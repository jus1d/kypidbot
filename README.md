# Heart Attack

## Quick Start

### Install deps

```bash
$ pip install -r ./requirements.txt
```

### Run

Rename `config.example.py` to `config.py` and add yout [API key](https://huggingface.co/docs/hub/en/security-tokens) for Hugging Face.

```bash
$ python3 ./matcher.py -o output.json input.json
```
